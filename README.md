# AI-Powered Resume Screening & Candidate Ranking System

## Overview

Recruiters often receive hundreds of resumes for a single job opening, making manual screening time-consuming and inconsistent. This project aims to streamline the initial hiring process by automatically analyzing resumes, extracting relevant skills, comparing candidates against job requirements, and generating candidate rankings based on role suitability.

The system combines Natural Language Processing (NLP) techniques with machine learning-based similarity scoring to identify the most relevant candidates while highlighting missing skills and qualification gaps.

---

## Project Objectives

* Automate resume screening and candidate shortlisting
* Extract technical and professional skills from resumes
* Compare candidate profiles with job descriptions
* Generate candidate ranking scores
* Identify skill gaps between candidates and job requirements
* Support data-driven hiring decisions

---

## Key Features

### Resume Parsing and Text Processing

* Resume text cleaning and preprocessing
* Removal of irrelevant characters and formatting
* Tokenization and normalization of text data

### Skill Extraction

* Identification of technical and professional skills
* Skill matching using predefined skill dictionaries
* Structured skill representation for analysis

### Candidate Ranking

* Comparison of resumes against job descriptions
* Similarity-based scoring mechanism
* Automated ranking of applicants

### Skill Gap Analysis

* Detection of missing skills
* Identification of improvement areas for candidates
* Generation of role-specific recommendations

### Hiring Insights

* Candidate comparison dashboards
* Skill distribution analysis
* Top candidate recommendations
